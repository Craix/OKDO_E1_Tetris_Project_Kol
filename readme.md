OKDO E1 Tetris

Module info: <br>
LCD 128 x 160 SPI ST7735S <br>
Thumb Joystick with button <br>

Connection schematic: <br>
TFT (FLXC_8  - SPI)
* LED - VDD (3.3V)
* SCK - PIO1_2 (SCK)
* SDA - PIO0_26 (MOSI)
* A0 -  PIO1_26
* RESET - PIO1_27
* CS - PIO1_1 (SSEL 1)

[ Joystick (ADC0 (N & P))] <br>
* VRX - PIO0_16
* VRY - PIO0_23 
* SW - PI0_15 (INT)

Common: <br>
* VDD - VDD(3.3V) <br>
* GND - GND
